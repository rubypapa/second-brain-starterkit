# 🧠 제2의 뇌 스타터킷

**AI(Claude)가 세션 끝나면 다 까먹는 문제를, 파일 몇 개로 끝냅니다.**
매번 처음부터 설명하고, 어제 정한 걸 잊고, 같은 실수를 반복하는 AI에게 — "제2의 뇌"(영속 기억)를 붙여주는 킷입니다.

> 강의 아니고, 그냥 복붙해서 따라 만드세요. 코딩 몰라도 됩니다.
> 유튜브 **[클로드 빌더 루비파파]** 에서 만드는 과정을 봅니다.

---

## ⬇️ 다운로드 → ⚡ 원클릭 설치 (만들기 힘든 분은 이것만)
1. 우측 상단 초록 **`Code → Download ZIP`** (또는 [📦 zip 받기](제2의뇌_스타터킷.zip)) → 압축 풀기
2. **Windows**: 폴더 안 **`설치.bat` 더블클릭**  /  **Mac·Linux**: 터미널에서 `bash install.sh`
3. 프로젝트 폴더 경로 + 이름만 입력하면 → 파일 복사·설정 자동. **끝.**
   (설치 후: 그 폴더에서 Claude Code 켜고, `CLAUDE.md`의 `[[ ]]`만 채우면 됨 — Claude에게 채워달라 해도 OK)
> ※ Python 필요(hook 실행용). 없으면 python.org에서 설치.

## 이게 뭘 하나 (3기둥)
| 기둥 | 파일 | 하는 일 |
|---|---|---|
| **① 색인** | `MEMORY.md` + `memory/` | 대화·결정·실수를 "사실 하나=파일 하나"로 쌓고 목차화. 흩어진 기억이 검색됨 |
| **② 규칙** | `CLAUDE.md` | AI가 반복하는 실수를 규칙으로 굳혀 박아둠. 다음엔 안 틀림 |
| **③ hook** | `.claude/settings.json` + `tools/session_start.py` | 새 세션마다 과거 기록을 **강제로** 읽게 함. 어제 걸 기억한 채 시작 |

## 5분 셋업
1. `CLAUDE.md` · `MEMORY.md` · `.claude/` · `tools/` 를 당신 프로젝트 폴더 루트에 복사.
2. `CLAUDE.md`의 `[[ ]]` 부분을 당신 상황으로 채움(당신이 누구인지, AI가 지킬 규칙).
3. `tools/session_start.py` 맨 위 `PROJECT_NAME` 을 당신 프로젝트 이름으로.
4. Claude Code를 그 폴더에서 켬 → 새 세션마다 hook가 과거를 자동 통독.
5. 끝. "이거 memory에 기록해"라고 시키면 기억이 파일로 쌓입니다.

## 쓰는 법 (매일)
- 중요한 결정·사실 → "memory에 파일로 남겨줘"
- AI가 실수 → "CLAUDE.md 규칙에 박아둬, 다신 이러지 말게"
- 새 세션 → hook가 알아서 과거를 읽음. 이어서 일하면 됨.
